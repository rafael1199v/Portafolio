Proyectos

Sistema gestor de la postulación docente

El proyecto tenía como finalidad mejorar el proceso de la postulación docente en instituciones educativas. Dirigido principalmente a las universidades.
Donde se usaron tecnologías como Angular para el frontend y .NET Core con C# para el backend

Detector de deepfakes

El objetivo de este proyecto era crear una herramienta para ayudar a las personas a detectar contenido fraudulento por medio de una aplicación web.
Se entrenó un modelo de inteligencia artificial con Teachable Machine y se creó el frontend con Angular y se clasificaba las imagenes desde un backend hecho con ExpressJS
